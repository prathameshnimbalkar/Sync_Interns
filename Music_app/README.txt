Music Web App (HTML-CSS-JS) 
This is a music web application built using HTML, CSS and JS. It allows users to explore and enjoy music. Enjoy a seamless listening experience with a music player.

Getting Started
To view the landing page, simply open the index.html file in a web browser.

Structure :
1) The index.html file is the main HTML file that serves as the entry point for the web app.

2) The css directory contains the CSS files responsible for styling the web app. The styles.css file is the main CSS file that imports additional CSS files if needed.

3) The Javascript file has all the scripts, JavaScript is a versatile language that allows you to write code in multiple lines for better readability and maintainability.
